Playmaker - Visual scripting for Unity

Made by www.hutonggames.com

Realize your creative vision without knowing how to program!

Quickly make gameplay prototypes, A.I. behaviors, animation graphs, interactive objects, in-engine cut-scenes, interactive walkthroughs and more...

Check out our online docs: https://hutonggames.fogbugz.com/default.asp?W1

Installation Notes:

To install Playmaker import Gizmos and PlayMaker folders from the unitypackage.
To also see samples, import PlaymakerSamples and Standard Assets folders.
